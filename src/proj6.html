ASCII Pet - JS Web App

Project Description

ASCII Pet is a small JavaScript web app that lets users care for a virtual pet made entirely out of ASCII art. Players can name their pet, feed it, pet it, and put it to sleep, while its health and happiness levels change over time. The project uses JavaScript to manage state, local storage to keep progress between sessions, and timed events to make the experience interactive and dynamic. This project gave me hands-on practice with DOM manipulation, event handling, and persistent data using the browser’s storage API
