CarDeal

Project Details

CarDeal is a responsive car sales website designed to make browsing vehicles simple and straightforward. The project was built with a focus on creating a clean and professional interface that would work well on both desktop and mobile devices. I handled both the layout and front-end styling using SCSS and Bootstrap.

Scope

The main goal of this project was to build a visually clear and easy-to-navigate site where users could view cars and key information without being overwhelmed. It focused on basic browsing and clear visual hierarchy rather than complex features.

Design Decisions

I used a minimal color palette with high contrast to keep the interface easy to read. The layout was structured around Bootstrap’s grid system, ensuring consistency across screen sizes. SCSS variables helped keep the design clean and reusable. Emphasis was placed on spacing, typography, and image presentation to make browsing smooth and intuitive.

Results

The final site is simple, responsive, and easy to navigate. It highlights key car details at a glance and presents a polished, professional layout. This project strengthened my ability to build clean front-end layouts and use SCSS and Bootstrap effectively.
