FLIXX – API Integration Project

Project Description

This project builds on the Flixx interface with a fully functional search experience that lets users filter between movies and TV shows. The results display dynamically with pagination and clear visual hierarchy, making it easy to browse through large sets of content. I focused on writing efficient JavaScript for API requests, managing user input, and presenting real-time search results in a structured, accessible way. This project helped me grow more comfortable working with APIs and building smooth, user-friendly features.
