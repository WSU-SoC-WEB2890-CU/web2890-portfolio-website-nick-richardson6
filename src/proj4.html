WCAG Accessibility Site

Project Details

This project is a responsive, accessibility-focused website designed to demonstrate how clear communication and inclusive design improve the user experience for everyone. It includes sections on accessibility best practices, plain language principles, and visual design choices. I handled all design and development aspects, including research, writing, and front-end implementation

Scope

The goal of this project was to create an informational site that showcases accessibility in action. It uses structured content, data visualizations, and examples of plain language to make complex topics approachable. The site also includes an accessibility statement, keyboard navigation support, and compatibility with assistive technologies

Design Decisions

I used a simple, high-contrast color palette, clear typography (Poppins), and semantic HTML to make the site easy to read and navigate. The layout follows a top-down reading flow supported by a responsive Bootstrap grid. Plain language was applied across all pages to keep the content clear and understandable. Accessibility features include alt text, skip navigation links, descriptive link text, and keyboard-friendly focus states

Results

The final prototype gave users a smooth and focused reading experience. Testers described it as easy to navigate and visually clean. This project helped me strengthen my ability to design with real user needs in mind.
